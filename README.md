# 💬 Real-Time Support Chat

## 📌 Project Overview
Real-Time Support Chat is a web-based chat application that enables instant communication between users and support agents. It uses Socket.IO for real-time messaging and Node.js with Express for the backend.

## ✨ Features
- Real-time messaging
- User-friendly interface
- Instant message delivery
- Multiple client support
- Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Socket.IO
- MongoDB

## 🚀 How to Run
1. Clone or download the repository.
2. Install dependencies using:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open your browser and access the application.
